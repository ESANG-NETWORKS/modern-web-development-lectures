# week1-1

## 주제

Git 기초와 프로젝트 킥오프를 VS Code 중심으로 진행하는 회차다.

## 진행 목표

- Source Control 화면에서 변경 사항을 읽는다.
- Stage, Commit, Push, Pull 흐름을 익힌다.
- 강의에서 사용할 프로젝트 배경과 목표를 이해한다.

## 권장 진행 순서

1. `docs/1주차-1_Git_기초_가이드.md`
2. `docs/KICK_OFF.md`
3. `docs/IDEA.md`
4. `docs/1주차_교육가이드_원본.md` (원문 참고가 필요할 때만)

## 폴더 구성

- `docs`: 1회차 강의 문서와 프로젝트 배경 자료
- `src`: 문서 중심 회차이므로 코드 대신 안내 문서 제공

## 메모

이 회차는 Git 사용 감각과 프로젝트 맥락을 맞추는 데 집중한다.
