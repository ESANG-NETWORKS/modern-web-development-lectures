# week1-2

## 주제

GitHub Flow, Pull Request, 리뷰와 병합을 실습하는 회차다.

## 진행 목표

- 이슈 단위 브랜치에서 작업을 만든다.
- PR을 생성하고 리뷰 코멘트를 반영한다.
- 병합 전략과 브랜치 정리 흐름을 이해한다.

## 권장 진행 순서

1. `docs/1주차-2_GitHub_Flow_PR_가이드.md`
2. `docs/KICK_OFF_v2.md`
3. `docs/archive/survey` 하위 자료 (선택)

## 폴더 구성

- `docs`: 2회차 강의 문서와 팀 운영 방식 참고 자료
- `src`: 문서 중심 회차이므로 코드 대신 안내 문서 제공

## 메모

이 회차는 Git 사용법보다 협업 규칙과 PR 품질 기준에 더 집중한다.
