# week3-1

## 주제

Vue 3 화면 구조와 상태 기반 UI를 교육용 스켈레톤으로 설명하는 회차다.

## 진행 목표

- Options API 기준으로 `data`, `computed`, `methods`, `mounted` 배치를 읽는다.
- 로그인, 목록, 상세, 등록 화면의 상태 흐름을 이해한다.
- OpenAPI 클라이언트와 화면 상태 연결 방식을 설명한다.
