# week2-1

## 주제

ASP.NET Core Web API 입문과 CLI 기반 프로젝트 조립을 실습하는 회차다.

## 진행 목표

- `dotnet new`로 만든 API 프로젝트 구조를 읽는다.
- `Program.cs`, DI, middleware, controller 흐름을 설명한다.
- OpenAPI와 Swagger UI를 연결해 API 테스트까지 완료한다.
